## React bolierplate app

TypeScript + React app boilerplate with bleeding edge dependencies.

### Features
- Webpack 4
    - Chunk split
    - Tree shaking
    - Dynamic imports
- Babel 7
- TypeScript 3
- SCSS
    - CSS modules
- React.js
- Redux
    - Redux thunk
    - Dev tools
- Reselect
    - Dev tools
