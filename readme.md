## React bolierplate app

TypeScript + React app boilerplate with bleeding edge dependencies.

### Features
- Webpack 4
    - Chunk split
    - Tree shaking
    - Dynamic imports
- Babel 7
- TypeScript 3
- SCSS
    - CSS modules
- React.js
- Redux (optional)
    - Redux thunk
    - Dev tools
- Reselect (optional)
    - Dev tools
